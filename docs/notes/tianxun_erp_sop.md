# 天勳 ERP 系統部署與權限設定標準作業流程 (SOP)

## 1. 系統登入與權限指派
* 所有新進員工須由人事部門填單申請 ERP 帳號。
* 帳號登入網址為：`http://erp.tianxun.com.tw:9000/`。
* 預設權限分為三種角色：
  * **系統管理員 (Admin)**：具備最高權限，可修改系統設定、備份資料庫。
  * **一般操作員 (User)**：可執行日常單據輸入、查詢報表。
  * **主管簽核 (Manager)**：具備簽核請購單、出貨單之核准權限。

## 2. ERP 備份規範
* 系統將於每日凌晨 02:00 自動執行完整備份 (Full Backup) 並存於 AWS S3。
* 系統管理員需於每週五下午 17:00 執行人工異地備份。

## 3. 常見故障處理
* 若遇到 "DB Connection Timeout" 錯誤，請先確認內部資料庫伺服器 (IP: 192.168.1.50) 是否正常運作。
